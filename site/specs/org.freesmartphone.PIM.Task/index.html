freesmartphone.org PIM.Task Interface

Description

This interface provides access to one PIM task.

org.freesmartphone.PIM.Task

  • GetMultipleFields
  • GetUsedBackends
  • Update
  • Delete

Signals

None

Methods

GetContent ( ) → a{sv}

Description: Get the fields of one task.

Returns:

a{sv}: task_data The task’s data.

GetMultipleFields ( s ) → a{sv}

Description: Get a specified field list of the task’s data.

Parameters:*

s: field_list The list of fields to get, split by “,” (coma) char.

Returns:

a{sv}: field_data The data of the requested fields.

GetUsedBackends ( ) → as

Description: Get used backends in task.

Returns:

as: backends The list of backends used in this task.

Update ( a{sv} )

Description: Update one or more fields of this task.

Parameters:*

a{sv}: task_data The list of fields/values to update for this task.

Description: Delete this task.
